# Homebrew Tap

A personal [Homebrew](https://brew.sh) tap containing custom formulas.

## Installation

```bash
brew tap jaspermayone/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [slack-profile](https://github.com/jaspermayone/slack-profile-cli) | CLI tool for updating Slack user profiles |
| [boxcar](https://github.com/jaspermayone/boxcar) | Rails app generator with opinionated defaults |

## Usage

### Install a formula

```bash
brew install jaspermayone/tap/<formula>
```

For example:

```bash
brew install jaspermayone/tap/boxcar
brew install jaspermayone/tap/slack-profile
```

### Update formulas

```bash
brew update
brew upgrade <formula>
```
